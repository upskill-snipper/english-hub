## How to use this file

For each named creator below:

1. Find the creator's segment (1–5).
2. Copy the **segment template** at the top of that segment.
3. Replace `[FIRST NAME]` and `[HOOK]` with the values supplied in the creator's row.
4. Replace `[CTA-URL]` with the UTMed link supplied in the segment template.
5. Send via the platform listed in the row (TikTok DM, Instagram DM, YouTube channel email if listed, etc.).
6. Update the tracker at the bottom.

**Sending pace:** 30–50 messages per platform per day for the first 3 days (TikTok throttles new accounts hard; Instagram less so but still). Then ramp by 10/day.
