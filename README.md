# The English Hub

Professional KS3 and GCSE English tutoring platform.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (database + auth)
- Stripe (payments)
- Vercel (hosting)

## Domain
https://theenglishhub.app

## Development
```bash
cp .env.example .env.local
# Fill in env vars
npm install
npm run dev
```
