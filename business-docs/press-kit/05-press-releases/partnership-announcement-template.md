# Press Release Template — Partnership Announcement

**How to use this template**

- A partnership release is co-authored. Never send it without the partner's written sign-off on the final text.
- Both organisations must approve each other's quotes *and* boilerplates.
- Keep the release under 450 words — partnership stories get cut aggressively by education desks.
- Save as `partnership-[YYYY-MM-DD]-[partner-slug].md` in the `sent/` subfolder.
