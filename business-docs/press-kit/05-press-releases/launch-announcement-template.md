# Press Release Template — Product Launch

**How to use this template**

- Replace every `[SQUARE-BRACKET PLACEHOLDER]` with a real value.
- Do not change the structure (headline → subhead → dateline → lede → body → quote → boilerplate → contact). UK education desks expect this shape.
- Keep the whole release under 500 words. Journalists skim; they do not wade.
- Always end with the 150-word boilerplate from `04-boilerplate/company-boilerplate.md`.
- Save as `launch-[YYYY-MM-DD]-[slug].md` in a new `sent/` subfolder when it ships.
- Log it in `09-tracking/press-outreach-log.csv`.
