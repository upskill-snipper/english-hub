# The English Hub — Logo Usage Guide

The English Hub logo is our most recognisable asset. This guide is mandatory reading for anyone placing our mark in print, on web, in a deck, on a school display, or in a press article.

---

## 1. The logo system

The logo has two components and three lockups:

**Components**
- **Wordmark** — "The English Hub" set in Mona Sans SemiBold, with bespoke kerning.
- **Icon** — a rounded square containing a stylised open-book monogram. The book opens toward an upward tick to suggest progress.

**Lockups**
1. **Horizontal primary** — icon on the left, wordmark on the right. This is the default. Use this 90% of the time.
2. **Stacked** — icon above, wordmark beneath. Use for square placements (Instagram profile, loyalty card, staff lanyard).
3. **Icon only** — the icon alone. Use *only* when space is too tight for the wordmark: favicons, app icons, social profile avatars at circles under 64 pixels, and as a repeat pattern or watermark.

Never recreate or rebuild the logo from screenshots.

## 3. Minimum size

| Context | Minimum width |
|---|---|
| Web, desktop | 120 px |
| Web, mobile | 96 px |
| Icon only on web | 24 px |
| Print | 24 mm |
| Icon only in print | 8 mm |
| Favicon | 16 × 16 px (use 32 × 32 source minimum) |

Below these sizes the logo loses legibility. Use the icon-only lockup instead.

---

## 4. Clear space

There must be a minimum clear space around the logo equal to the **height of the "E"** in the wordmark (`1×`), on all four sides. Nothing may intrude into this space — no text, no images, no edges of photographs, no page margins.

When stacking the logo vertically (for example in a footer column), use `1.5×` above and below to create breathing room.

---

## 5. Backgrounds

| Background | Use this logo |
|---|---|
| Ink (`#0B0F17`) — our primary dark | Light variant, Hub Emerald accent |
| White or very light (`#F1F4F9`+) | Dark variant |
| Hub Emerald (`#22C58A`) | White one-colour variant |
| Mid-tone grey or busy colour photograph | Place over a black or white scrim set to 60–80% opacity |
| Photographs of students | Always add a scrim. Never place the logo directly over faces or text |

---

## 6. Wordmark vs. icon — when to use which

| Use the wordmark (with icon) when… | Use the icon alone when… |
|---|---|
| First impression — homepage hero, cover slide | Repeat branding — favicon, app tile, social avatar |
| Press releases and media | Watermark on screenshots and videos |
| Printed materials with room to breathe | Product UI chrome under 32px |
| Co-branding with a partner | Pattern or background texture |

---

## 7. Co-branding (with schools, partners, investors)

When our logo appears next to another organisation's:

1. Use the horizontal primary lockup.
2. Separate the two marks with a vertical rule at 1px, in neutral grey, the same height as the logos.
3. Keep the heights visually equal — the x-height of both wordmarks should match, not the absolute pixel height.
4. Honour the other organisation's own logo rules (clear space, colour) as if they were ours.
5. Never merge, overlap, or recolour the partner logo to match ours.

---

## 8. What you must not do

Do not:

- **Stretch or squash** the logo. Lock the aspect ratio.
- **Rotate** the logo. It is always horizontal (or stacked for the stacked lockup).
- **Recolour** the logo. Use only the approved variants.
- **Add effects** — no drop shadows, no glows, no bevels, no gradients over the mark itself.
- **Place text inside** the clear space.
- **Reproduce below the minimum size.**
- **Screenshot** the logo from the live site and re-use it as an asset. Always use the vector source.
- **Enclose** the logo in a second shape (a circle, a pill, a frame) unless it is the icon-only lockup.
- **Use the old logo** if we ever rebrand. Archive old files with `_DEPRECATED_` prefix.
