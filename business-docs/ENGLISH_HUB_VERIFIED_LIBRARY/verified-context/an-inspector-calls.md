# An Inspector Calls -- Verified Context
Author: J.B. Priestley (1894-1984). Written 1945, set 1912. First performed USSR 1945, UK 1946.

Historical context: Written at end of WW2 -- Priestley had seen two world wars that might have been avoided if society acted more responsibly. Set in 1912 gives dramatic irony (audience knows WW1 and WW2 are coming). Titanic sank April 1912 (Birling's speech about the Titanic 'absolutely unsinkable').

Literary context: Well-made play structure with revelation chain. Priestley socialist; Birling embodies irresponsible capitalism. Inspector Goole = 'Ghoul'. Each character represents a different moral stance.

Themes: Social responsibility; class; gender inequality; capitalism vs socialism; collective vs individual morality.

Exam: AQA GCSE Set Text Paper 2. Compare: A Christmas Carol (social responsibility), Macbeth (moral corruption).
