# Pearson Endorsement — Requirements Checklist

**Purpose:** A single consolidated checklist of everything we believe Pearson Publisher Relations will want to see, in the order they are likely to ask for it. Tick as each item is confirmed complete and reviewed by the founder.

**Status legend:** `[ ]` not started · `[~]` drafted / in progress · `[x]` ready for submission

Last reviewed: 2026-04-10

---

## A. Publisher identity and credibility

- [ ] Registered UK company with Companies House number
- [ ] Registered office address
- [ ] VAT number (if registered)
- [ ] Directors listed with brief bios
- [ ] Ultimate beneficial ownership disclosed
- [ ] Insurance: public liability and professional indemnity (minimum GBP 2M each — confirm with Pearson)
- [ ] Data protection registration (ICO)
- [ ] Years trading and evidence of going concern
- [ ] Any previous endorsement relationships with other awarding bodies (AQA, OCR, WJEC, SQA, CIE) disclosed

## B. Qualification scope

For each specification being endorsed against, confirm:

- [ ] **Edexcel GCSE English Literature (1ET0)** — version / issue number referenced
- [ ] **Edexcel GCSE English Language (1EN0 / 1EN2)** — spec version confirmed
- [ ] **Edexcel International GCSE English Literature (4ET1)** — Issue 3 (first assessment May/June 2026)
- [ ] **Edexcel International GCSE English Language A (4EA1)** — Issue 7 (from 1 Sept 2025)
- [ ] Current text of the relevant specification has been obtained directly from qualifications.pearson.com (not third-party copies)

## C. Product description

- [ ] Product name and version
- [ ] Format (digital subscription / printed book / hybrid)
- [ ] Access model (licence type, per-student or per-teacher)
- [ ] Geographic availability
- [ ] Pricing and, where relevant, trial access
- [ ] Target audience (Key Stage, age range, teacher vs student vs parent)
- [ ] Word count and number of teaching hours supported
- [ ] Screenshots or sample pages (minimum 10 pages / screens representative of the product)

## D. Curriculum alignment

- [ ] Spec-by-spec mapping from every content statement in the Edexcel specification to The English Hub content (see `02-application/curriculum-alignment-gcse.md` and `curriculum-alignment-igcse.md`)
- [ ] Gaps in coverage flagged honestly
- [ ] Roadmap to close gaps with target dates (if any)

## E. Assessment objective alignment

- [ ] Assessment objectives listed and weighted per Edexcel spec
- [ ] Evidence of how each AO is practised in The English Hub content
- [ ] Clear statement that publisher-written practice questions are **not** "approved by Pearson"
- [ ] Marking guidance that references Edexcel level descriptors honestly without quoting them verbatim without permission

## F. Editorial quality and authorship

- [ ] Named subject lead / series editor
- [ ] Named contributing authors with qualifications and experience
- [ ] Evidence that at least some contributors are current or recent Pearson Edexcel examiners (see `03-evidence/examiner-credentials.md`)
- [ ] Editorial workflow documented (`03-evidence/editorial-process.md`)
- [ ] Accuracy policy documented (`03-evidence/accuracy-commitment.md`)
- [ ] Quotation permissions documentation (texts on the Edexcel anthology / set text lists)
- [ ] Peer review / second reader evidence

## G. Safeguarding and inclusivity

- [ ] Safeguarding policy (`03-evidence/safeguarding-compliance.md`)
- [ ] Age-appropriate content review
- [ ] Diversity, equity and inclusion review
- [ ] Accessibility statement aligned to WCAG 2.2 AA
- [ ] Content warnings process for mature / sensitive texts
- [ ] DSL (designated safeguarding lead) named, if relevant to the platform design

## H. Legal, trademark and IP

- [ ] Terms of service reviewed for any claims about exam boards
- [ ] No current use of "Pearson", "Edexcel" or "Endorsed" that would pre-empt the endorsement
- [ ] IP ownership of all original content asserted (`04-legal/ip-considerations.md`)
- [ ] Licences for quoted set texts confirmed with rights holders
- [ ] Draft trademark use agreement notes prepared (`04-legal/trademark-use.md`)

## I. Commercial

- [ ] Royalty structure modelled (`05-commercial/royalty-structure.md`)
- [ ] Draft commercial terms prepared (`05-commercial/commercial-terms-proposal.md`)
- [ ] Finance contact named
- [ ] Reporting cadence (monthly / quarterly royalty statements)

## J. Sample content submission pack

- [ ] Cover letter (`02-application/cover-letter-draft.md`)
- [ ] Company profile (`02-application/company-profile.md`)
- [ ] Product overview (`02-application/product-overview.md`)
- [ ] GCSE alignment (`02-application/curriculum-alignment-gcse.md`)
- [ ] IGCSE alignment (`02-application/curriculum-alignment-igcse.md`)
- [ ] Assessment alignment (`02-application/assessment-alignment.md`)
- [ ] Sample pages document (`02-application/sample-content-submission.md`)
- [ ] Editorial process note
- [ ] Accuracy commitment
- [ ] Safeguarding statement
- [ ] Trademark use note (our position)

## K. Programme management

- [ ] Application tracker up to date (`06-tracking/application-tracker.md`)
- [ ] Contacts log up to date (`06-tracking/pearson-contacts.md`)
- [ ] Internal accountable owner named (one person)
- [ ] Decision gate for "withdraw if uneconomic" pre-agreed with board

## L. Things we DO NOT need (but are often confused with this process)

- [ ] Ofqual regulatory registration — not required, this is a publisher endorsement not a qualifications regulator process
- [ ] DfE listing — not applicable
- [ ] ISBN for digital subscription — not required unless a physical print edition is also produced
- [ ] Exam centre status — not applicable, The English Hub is not an exam centre
